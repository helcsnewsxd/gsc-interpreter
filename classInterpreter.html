Detailed Description

The Interpreter class evaluates expressions in the GSC language.

Note
It implements the ExprVisitor interface to visit different types of expressions and evaluate them accordingly. The results of the evaluations are returned as std::any types, allowing for flexibility in the types of values that can be returned.
